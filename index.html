 MANUAL DE INSTALAÇÃO

Siga os passos abaixo para efetuar a instalação do sistema:

1 - descompactar o arquivo SGS.zip na raiz do seu servidor;

2 - criar um banco de dados chamado sgs;

3 - importar o script sgs.slq da pasta Banco de Dados;

4 - Acessar via navegador o seguinte endereço http://localhost/sgs/;

5 - logar com o usuário: josni e senha: 12345678 para acessar administrador;

6 - navegue a vontade;
